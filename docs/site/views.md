GitVista is organized around orientation first. The graph gives branch shape, the repository overview summarizes the current state, and the file and diff views let you drill down without losing context.

- Graph view shows commit flow, merges, and branch movement.
- Repository view highlights branch state, HEAD, remotes, tags, and overall repository metadata.
- File explorer, blame, and diff views let you move from topology to exact file changes in one session.
- Hosted repo URLs preserve the current repository, and hosted commit permalinks preserve the selected commit.
