This skill guides creation of distinctive, production-grade frontend interfaces that avoid generic "AI slop" aesthetics. Implement real working code with exceptional attention to aesthetic details and creative choices.

The user provides frontend requirements: a component, page, application, or interface to build. They may include context about the purpose, audience, or technical constraints.

## Design Thinking

Before coding, understand the context and commit to a bold aesthetic direction:
- **Purpose**: Identify what problem the interface solves and who uses it.
- **Tone**: Pick a clear design direction. Examples: brutally minimal, maximalist chaos, retro-futuristic, organic/natural, luxury/refined, playful/toy-like, editorial/magazine, brutalist/raw, art deco/geometric, soft/pastel, industrial/utilitarian.
- **Constraints**: Capture technical requirements such as framework, performance, and accessibility.
- **Differentiation**: Define one unforgettable element the user will remember.

Choose a clear conceptual direction and execute it with precision. Bold maximalism and refined minimalism both work when the design is intentional.

Then implement working code (HTML/CSS/JS, React, Vue, etc.) that is:
- Production-grade and functional
- Visually striking and memorable
- Cohesive with a clear aesthetic point of view
- Meticulously refined in detail

## Frontend Aesthetics Guidelines

Focus on:
- **Typography**: Choose distinctive fonts that elevate the interface. Avoid generic families like Arial, Inter, Roboto, and default system stacks. Pair expressive display typography with a refined body face.
- **Color and Theme**: Commit to a cohesive direction. Use CSS variables. Favor dominant colors with sharp accents over timid, evenly distributed palettes.
- **Motion**: Use purposeful animation. Prefer CSS-only animation for plain HTML and use motion libraries for React when available. Prioritize high-impact moments such as page-load choreography with staggered reveals.
- **Spatial Composition**: Use layouts with intention: asymmetry, overlap, diagonal flow, grid breaks, and deliberate density or negative space.
- **Backgrounds and Detail**: Build atmosphere with gradients, textures, patterns, layered transparencies, dramatic shadows, decorative borders, grain overlays, and other context-appropriate details.

Avoid generic AI aesthetics:
- Overused default font families
- Cliched color schemes, especially purple-on-white gradients
- Predictable layouts and cookie-cutter component patterns

Interpret creatively and make unexpected choices suited to the task context. Vary themes, type families, and visual language across outputs.

Match implementation complexity to the chosen aesthetic vision. Maximalist concepts require deeper visual systems and richer animation; minimalist concepts require precision, restraint, and careful spacing decisions.
